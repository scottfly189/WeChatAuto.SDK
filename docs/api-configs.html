配置类 (Configs)

WeAutoCommon 库中的配置类，用于配置微信自动化的各种选项。

WeChatConfig

微信自动化参数配置类，包含所有可配置的选项。

属性

属性 类型 默认值 说明
ApiKey string "" 接口KEY
DefaultSavePath string wxauto_download 下载文件/图片默认保存路径
CheckAppRunningInterval int 3 检查微信客户端是否运行的时间间隔（秒）
AppRetryWaitTime int 10 微信客户端退出时的重试等待时间（秒）
EnableCheckAppRunning bool true 是否启用检查微信客户端是否运行
ListenInterval int 5 监听消息时间间隔（秒）
MomentsListenInterval int 10 朋友圈监听时间间隔（秒）
NewUserListenerInterval int 5 新用户监听时间间隔（秒）
MonitorSubWinInterval int 5 监听子窗口时间间隔（秒）
DebugMode bool false 是否启用调试模式
CaptureUIPath string Captures 出错后捕获UI保存路径
EnableRecordVideo bool false 是否启用视频录制
TargetVideoPath string Videos 视频录制保存路径
EnableMouseKeyboardSimulator bool false 是否启用鼠标键盘模拟器
KMDeiviceVID int 0x2612 键鼠模拟器设备VID
KMDeivicePID int 0x1701 键鼠模拟器设备PID
KMVerifyUserData string "" 键鼠模拟器验证用户数据
OffsetOfClick int 5 点击偏移量（像素）
OutputStringType int 4 输出字符串编码类型
MouseMoveMode int 0 鼠标移动模式
ProcessDpiAwareness int 1 进程DPI感知值（0:不设置, 1:系统DPI感知, 2:每显示器DPI感知）

使用示例

// 初始化时配置
WeAutomation.Initialize(services, options =>
{
    options.DebugMode = true;
    options.EnableRecordVideo = true;
    options.TargetVideoPath = @"C:\Videos";
    options.ListenInterval = 2; // 2秒检查一次消息
    options.EnableMouseKeyboardSimulator = false;
});

// 运行时访问配置
var config = WeAutomation.Config;
if (config.DebugMode)
{
    Console.WriteLine("调试模式已启用");
}

WeMettingConfig

会议配置类，用于微信会议相关的配置。

Language

语言配置静态类。

属性

  • CurrentLanguage - 当前语言（默认 Chinese）
  • Chinese - 中文常量 "Cn"
  • ChineseTraditional - 繁体中文常量 "CnT"
  • English - 英文常量 "En"
