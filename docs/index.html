WeChatAuto.SDK

WeChatAuto.SDK 是一个基于 UI 自动化技术的微信 PC 客户端自动化开发框架。 它提供了完整的微信自动化功能，支持消息发送、接收、群聊管理、朋友圈操作等，并优先支持人工智能接入。

项目架构

WeChatAuto.SDK 采用模块化设计，包含三个核心库：

WeAutoCommon - 基础公共库

提供微信自动化所需的核心类型定义、模型、枚举、接口和工具类。 包括聊天类型、消息类型、会话模型、结果封装等基础组件。

WeChatAuto - 微信自动化核心库

提供完整的微信 PC 客户端自动化功能实现。包含微信客户端工厂、主窗口组件、 消息发送器、聊天内容区管理等核心功能。

WeChatAuto.MCP - MCP Server 支持库

基于 Model Context Protocol (MCP) 的微信自动化服务器实现。 提供标准化的 MCP 接口，使得任何支持 MCP 协议的客户端都可以调用微信自动化功能。

快速开始

只需几个步骤即可开始使用 WeChatAuto.SDK：

using Microsoft.Extensions.DependencyInjection;
using WeChatAuto.Components;
using WeChatAuto.Services;

// 1. 初始化服务
var services = new ServiceCollection();
WeAutomation.Initialize(services);

// 2. 获取微信客户端
var factory = services.BuildServiceProvider()
    .GetRequiredService<WeChatClientFactory>();
var client = factory.GetWeChatClient("微信昵称");

// 3. 发送消息
await client.SendWho("好友名称", "你好！");
