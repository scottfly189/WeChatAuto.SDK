组件 API

本文档详细介绍了 WeChatAuto.SDK 中所有组件的 API。点击左侧菜单查看各个组件的详细说明。

核心组件

WeChatClient

微信客户端，表示一个微信实例。提供消息发送、接收、群聊管理等核心功能。

WeChatClientFactory

微信客户端工厂，管理多个微信客户端实例，支持多微信实例管理。

WeChatMainWindow

微信主窗口，封装微信窗口的所有组件，包括工具栏、导航栏、搜索等。

ChatContent

聊天内容区，表示一个聊天窗口，包含聊天头部和聊天主体。

ChatBody

聊天主体，包含消息气泡列表和发送器，提供消息监听功能。

Sender

消息发送器，负责发送消息、文件、表情等。

窗口和列表组件

  • SubWinList - 子窗口列表，管理所有弹出子窗口
  • SubWin - 子窗口，表示微信的弹出窗口
  • ConversationList - 会话列表，管理微信主窗口左侧的会话列表
  • AddressBookList - 通讯录列表，管理通讯录功能
  • MessageBubbleList - 消息气泡列表，管理聊天中的所有消息气泡
  • MessageBubble - 消息气泡，表示一条消息
  • ChatHeader - 聊天头部，包含聊天标题和信息按钮

功能组件

  • Navigation - 导航栏，提供微信主窗口的导航功能
  • Moments - 朋友圈，管理朋友圈相关功能

工具类

工具类 (Utils) - WeChatAuto 库中的工具类集合，包括：

  • AutoLogger - 自动日志记录器
  • WeChatCaptureImage - 微信截图
  • WeChatRecordVideo - 微信视频录制
  • WxLocator - 微信定位器
  • MessageBubbleParser - 消息气泡解析器
  • MomentsHelper - 朋友圈助手
  • 以及其他辅助工具类

公共类

数据模型

数据模型 (Models) - WeAutoCommon 库中的数据模型：

  • Result<T> - 结果封装类
  • Maybe<T> - 可选值类型
  • Conversation - 会话模型
  • ChatResponse - 聊天响应
  • MomentItem - 朋友圈项
  • 以及其他数据模型

枚举类型

枚举类型 (Enums) - 所有枚举类型定义：

  • ChatType - 聊天类型
  • MessageType - 消息类型
  • MessageSourceType - 消息来源类型
  • NavigationType - 导航类型
  • 以及其他枚举类型

配置类

配置类 (Configs) - 配置选项：

  • WeChatConfig - 微信配置
  • WeMettingConfig - 会议配置

公共工具类

公共工具类 (Common Utils) - WeAutoCommon 库中的工具类：

  • UIThreadInvoker - UI线程调用器
  • RandomWait - 随机等待
  • EmojiListHelper - 表情列表助手
  • IMEHelper - 输入法助手
  • 以及其他工具类
