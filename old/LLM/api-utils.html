工具类 (Utils)

WeChatAuto 库中的工具类集合，提供各种辅助功能。

AutoLogger<T>

自动日志记录器，提供类型化的日志记录功能。

使用示例

var logger = serviceProvider.GetRequiredService<AutoLogger<MyClass>>();
logger.Trace("跟踪信息");
logger.Debug("调试信息");
logger.Info("信息");
logger.Warn("警告");
logger.Error("错误");
