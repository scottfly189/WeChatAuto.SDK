# WeChatAuto.SDK 文档

这是 WeChatAuto.SDK 的完整 HTML 使用指南文档。

## 文档结构

- **index.html** - 首页，介绍 SDK 的特性和架构
- **getting-started.html** - 快速上手指南
- **basics.html** - 基础使用指南，深入介绍核心概念
- **api.html** - 完整的组件 API 文档
- **styles.css** - 统一的样式文件

## 架构图

文档中引用了架构图 `architecture-diagram.png`。请将 WeChatAuto.SDK 类关系图保存为此文件名，放在 docs 目录下。

如果暂时没有图片，可以：
1. 使用占位图片
2. 或者注释掉图片引用，保留文字说明

## 使用方法

直接在浏览器中打开 `index.html` 即可查看文档。所有页面都使用相对路径，可以部署到任何 Web 服务器。

## 样式说明

文档样式参考了 Vue3 官方文档的设计：
- 左侧固定侧边栏导航
- 右侧主内容区域
- 响应式设计，支持移动端
- 代码高亮支持
- 清晰的层次结构
